## 3. 课前准备

**执行人**：全体人员 | **时间**：开营前 60 分钟

### 3.1 场地布置

- [ ] 室内场地：每桌 4 人，椅子 4 把，桌面清空
- [ ] 室外发射区：划定 30m×10m 安全区域，拉警戒线或贴地标
- [ ] 观众区：发射区后方 10m 外设置观众线
- [ ] 签到区：入口处设置签到桌，准备签到表 + 姓名贴

### 3.2 设备检查

- [ ] 电动发射台：通电测试，脚踏开关响应正常
- [ ] 电动气泵：压力达 4bar 自动停机，手动泄压阀正常
- [ ] 倒计时音响：蓝牙连接手机，音量适中
- [ ] 高度计：电池有电，与每组编号绑定
- [ ] PPT/视频：投影仪或大屏播放正常
- [ ] 计时器：手机或独立计时器准备到位

**主讲操作**：

| 时间 | 操作 | 话术/内容 |

## 5. 一级火箭拼装（30min）

**目标**：学生独立完成一级火箭组装，建立成功体验

### 5.1 拼装演示（5min）

**主讲操作**（配合大屏播放拼装教程视频）：

> 视频播放同步口播：
> "第一步：拿出你们桌面上的 600ml 瓶子，这就是火箭的身体。"
> "第二步：把 3D 打印的连接座套在瓶口往下推，卡紧。"
> "第三步：拿出三片尾翼，注意缺口方向一致，对准连接座的槽插进去。"
> "第四步：从头锥的侧面把螺丝穿进去，用螺母在另一侧旋紧。"
> "注意：每步做完举手示意助教老师检查——通过后才能走下一步。"

### 5.2 学生拼装（20min）

**主讲操作**：
- 巡回各组，拍照记录
- 每 5 分钟播报节奏："完成连接座的同学举手！""完成尾翼的同学举手！"

**助教 B 巡场检查要点**：
- [ ] 连接座是否卡到位（推到底，不松动）
- [ ] 尾翼方向是否一致（三片在同一平面）
- [ ] 头锥是否锁紧（螺丝旋紧，头锥不晃动）
- [ ] 瓶身是否有漏装步骤

**助教 C**：
- 发现普遍问题，及时口头提醒全体
- 发放加水量杯（装水环节统一进行）

### 5.3 装水 + 装发射塞（5min）

**主讲操作**：
> "现在所有小组都拼好了。接下来最后两步："
> "**装水**：用量杯装 200ml 水，用漏斗倒进瓶子。注意水不要洒到外面。"
> "**装发射塞**：把硅胶塞用力按进瓶口，确保不漏气。按不动的可以请旁边的助教老师帮忙。"

**助教 B 检查**（关键质检点）：
- [ ] 水量 200ml ± 20ml
- [ ] 硅胶塞完全塞入，气管接头朝上
- [ ] 倒置瓶子不滴水（验证密封性）

**助教 A**：
- 确认发射台就绪
- 检查气泵压力表

## 6. 一级火箭发射（30min）

**目标**：确保每组至少成功发射一次，建立信心

### 6.1 转移至发射区（5min）

**主讲操作**：
> "现在请各组队长拿着火箭，跟着助教老师到发射区。其他同学安静排队走过去。"
> "到达后站在观众线后面，不要越过黄线。"

**助教 B**：
- 带领各组有序前往发射区
- 每 2 组为一批次，其他组在观众区等候

### 6.2 发射流程（每组约 3min，共 8 组 → 20min）

**注意**：一级火箭结构简单，不设汇报检查点，助教 A 直接操作发射。

**分组发射流程**：

| 步骤 | 操作人 | 操作 | 话术 |
|------|--------|------|------|
| 1 | 助教 A | 接过火箭，倒置安装在发射台上，气管接头对准卡口 | — |
| 2 | 主讲 | 确认安全 | "发射区清场——所有人都退到观众线后面了吗？" |
| 3 | 助教 A | 启动气泵加压至 3bar | — |
| 4 | 主讲 | 倒计时 | "准备好——3——2——1——发射！"（音响播放倒计时音效） |
| 5 | 助教 A | 踩下脚踏开关 | — |
| 6 | 主讲 | 宣布结果 | "X 组火箭发射成功！高度：XX 米！"（读高度计数据） |
| 7 | 助教 C | 记录成绩到记录表 | — |

### 6.3 一级火箭发射后处理

发射完成后，所有组返回室内教室。

**助教 A**：
- 回收所有一级火箭，放置于回收区
- 检查瓶体是否有裂纹，有裂纹的报废处理

**助教 C**：
- 提醒学生喝水、上厕所（5min 休息）

## 7. 体育环节（30min）

**目标**：动静结合，释放体力，为二级火箭拼装蓄能

**活动建议**（三选一，提前确认场地与器材）：

| 活动 | 时长 | 所需器材 | 适合场地 |
|------|------|----------|----------|
| 花式跳绳 | 25min | 跳绳 8-10 根 | 操场/空地 |
| 飞盘九宫格 | 25min | 飞盘 4-6 个 + 九宫格架 | 操场 |
| 趣味接力赛 | 25min | 接力棒 4 根 | 操场/跑道 |

**操作说明**：
- 主讲老师带队到体育场地
- 助教 B 协助组织活动
- 助教 A + 助教 C 留在教室：回收一级火箭器材、按组摆放二级火箭物料

**体育环节结束前 5min，助教 C 提前回到教室，确认二级火箭物料全部到位。**

## 8. 二级火箭拼装（35min）

**目标**：制造更复杂的二级火箭，增加分离舱结构

### 8.1 二级火箭新结构说明（5min）

**主讲操作**（配合大屏播放二级火箭介绍视频）：
> "同学们，刚才我们成功发射了一级火箭。现在我们要做一枚更酷的——**两级火箭**！"
> "它比一级多一个**分离舱**——飞到半空中后，上半部分会弹出来，像真火箭一様！"
> "但难度也增加了——多了一个分离机构，装配要更细心。"

### 8.2 拼装演示（5min）

播放拼装教程视频，主讲同步口播：

> "第一步：两个瓶子——一个做主箭体，一个做分离舱（切掉底部做头锥罩）。"
> "第二步：在分离舱底部装上分离机构（卡扣朝外），弹簧放在中间。"
> "第三步：尾翼安装方法与一级相同。"
> "第四步：级间连接器套在主箭体瓶口，拧紧。"
> "第五步：分离舱对准级间连接器，卡扣卡紧——听到'咔嗒'一声才算到位。"

### 8.3 学生拼装（20min）

**各班拼装注意事项**：
- 分离机构卡扣必须卡紧（助教 B 逐组检查）
- 密封性比一级更高——硅胶塞务必按到底
- 瓶身标签建议撕掉（助教 C 准备吹风机或温水协助）

**助教 B 巡场检查要点**：
- [ ] 分离舱底部安装方向正确
- [ ] 弹簧位置居中，不被卡住
- [ ] 级间连接器旋紧不漏气
- [ ] 尾翼三片角度均匀

### 8.4 装水 + 装发射塞（5min）

**主讲**：
> "装水 200ml 到主箭体。分离舱不装水。装发射塞——注意这次要塞得更紧。"

**助教 B（关键质检点）**：
- [ ] 主箭体水量 200ml ± 20ml
- [ ] 分离舱不进水（如已进水，倒空擦干）
- [ ] 硅胶塞完全密封

## 9. 二级火箭发射（40min）

**目标**：所有组完成二级火箭发射，记录高度，评出冠军

### 9.1 转移至发射区（5min）

同 6.1，注意移动时二级火箭更重，提醒学生双手持握。

### 9.2 发射前汇报检查点（10min）

**二级火箭专属环节**——这是半日营的"质量控制点"。

**操作**：每组由主讲或助教 A 进行发射前检查：

| 检查项 | 标准 | 判定 |
|--------|------|------|
| 分离机构卡扣 | 卡到位，轻拉不脱 | □ 通过 □ 不通过 |
| 级间连接器 | 旋紧，无明显缝隙 | □ 通过 □ 不通过 |
| 密封性 | 倒置不滴水 | □ 通过 □ 不通过 |
| 尾翼角度 | 三片均匀，不歪斜 | □ 通过 □ 不通过 |
| 压力限值 | 二级火箭限压 4bar | □ 通过 □ 不通过 |

> 未通过检查的组，由助教 B 现场指导修正后重新排队。

### 9.3 发射流程（每组约 3min → 20min）

流程同一级火箭（见 6.2），但有以下差异：

- **压力提升至 4bar**（助教 A 确认压力表）
- **倒计时增加仪式感**：音响播放完整倒计时音频 + 光效配合
- **发射后观察分离**：主讲用话筒解说分离情况："看！分离舱弹开了——二级火箭分离成功！"
- **记录高度**：助教 C 读取高度计并记录

### 9.4 成绩汇总（5min）

**助教 C**：
- 汇总所有组成绩，排出名次
- 将排名表交给主讲

## 11. 附录

### 附录 A：常见问题与应急处理

| 问题 | 原因 | 处理方式 |
|------|------|----------|
| 发射塞漏气，压力上不去 | 硅胶塞未塞紧 | 重新用力按压硅胶塞，或更换新塞 |
| 火箭发射后原地不动 | 气压不足 / 水量过多 | 检查压力表是否达到限压值；减少水量至 150ml 重试 |
| 二级火箭未分离 | 卡扣太紧 / 弹簧失效 | 发射前用力按压卡扣测试弹力；如不分离仍可计入成绩，不属于失败 |
| 发射后瓶子爆裂 | 瓶体老化 / 压力过高 | 立即疏散，助教 A 关闭气泵；检查该瓶批次，同批次瓶全部更换 |
| 下雨天 | 天气变化 | 转入预案：发射区移至有顶棚区域；如无合适场地，延期发射至室内原理展示 |
| 学生突发身体不适 | — | 助教 C 立即带学生至休息区，联系家长 |

### 附录 B：发射台操作说明

**电动发射台操作流程**：

1. **开机检查**：接通电源 → 确认红色指示灯亮起 → 确认气泵气管连接牢固
2. **安装火箭**：将火箭倒置 → 发射塞气管接头对准发射台气管卡口 → 用力下压至卡紧
3. **加压**：按下气泵启动按钮 → 观察压力表上升 → 一级火箭达 3bar 停机，二级火箭达 4bar 停机
4. **发射**：确认发射区清场 → 倒计时 3 秒 → 踩下脚踏开关 → 电磁阀打开 → 火箭发射
5. **泄压**：发射完成后 → 按手动泄压阀释放管路残余压力 → 取下火箭

**注意**：
- 气泵不可长时间连续运转，连续发射 3 次后暂停 2 分钟散热
- 脚踏开关线缆需放置于显眼位置，避免绊倒
